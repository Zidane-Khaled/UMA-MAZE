# Uma Maze

Uma Maze is a simple fan-made maze game inspired by **Umamusume**.

In this game, you play as a character trapped inside a maze while being chased by **Machikane Tannhauser (Mambo)** and **Gold Ship**. Your goal is to navigate the maze and avoid getting caught.
